# Phase 1

Phase 1 establishes the Zodiacs SDK workspace, official Zodiacs.org registry,
market adapter contracts, React hook scaffolds, and minimal premium UI
components.

The registry is local source-controlled data. Apps do not scrape Zodiacs.org at
runtime. Solana SPL mints are modeled as native origins, and Base ERC-20
addresses are modeled as official bridged representations.
